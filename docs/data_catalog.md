# Каталог данных для золотого слоя

## Обзор
Уровень Gold - это представление данных бизнес-уровня, структурированное для поддержки аналитических и отчетных вариантов использования. Он состоит из **dimension tables** and **fact tables** для конкретных бизнес-показателей.

---

### 1. **gold.dim_customers**
- **Цель:** Хранит информацию о клиентах
- **Столбцы:**

| Имя столбца      | Тип данных     | Описание                                                                                     |
|------------------|---------------|-----------------------------------------------------------------------------------------------|
| customer_key     | INT           | Суррогатный ключ, однозначно идентифицирующий каждую запись о клиенте в таблице измерений.               |
| customer_id      | INT           | Уникальный цифровой идентификатор, присваиваемый каждому клиенту.                                        |
| customer_number  | NVARCHAR(50)  | Буквенно-цифровой идентификатор, представляющий клиента, используемый для отслеживания и привязки к нему.|
| first_name       | NVARCHAR(50)  | Имя клиента, записанное в системе.                                                                       |
| last_name        | NVARCHAR(50)  | Фамилия или отчество клиента.                                                                            |
| country          | NVARCHAR(50)  | Страна проживания клиента (например, "Австралия").                                                       |
| marital_status   | NVARCHAR(50)  | Семейное положение клиента (например, "Женат", "Холост").                                                |
| gender           | NVARCHAR(50)  | The gender of the customer (e.g., 'Male', 'Female', 'n/a').                                              |
| birthdate        | DATE          | The date of birth of the customer, formatted as YYYY-MM-DD (e.g., 1971-10-06).                           |
| create_date      | DATE          | The date and time when the customer record was created in the system                                     |

---

### 2. **gold.dim_products**
- **Цель:** Предоставляет информацию о продуктах и их свойствах.
- **Столбцы:**

| Имя столбца         | Тип данных     | Описание                                                                                     |
|---------------------|---------------|-----------------------------------------------------------------------------------------------|
| product_key         | INT           | Суррогатный ключ, однозначно идентифицирующий каждую запись продукта в таблице измерений продукта.                        |
| product_id          | INT           | Уникальный идентификатор, присвоенный продукту для внутреннего отслеживания и привязки к нему.                            |
| product_number      | NVARCHAR(50)  | Структурированный буквенно-цифровой код, представляющий продукт, часто используемый для категоризации или инвентаризации. |
| product_name        | NVARCHAR(50)  | Описательное название продукта, включающее ключевые сведения, такие как тип, цвет и размер.                               |
| category_id         | NVARCHAR(50)  | Уникальный идентификатор категории продукта, привязанный к его высокоуровневой классификации.                             |
| category            | NVARCHAR(50)  | Более широкая классификация продукта (например, велосипедов, компонентов) для группировки связанных товаров.              |
| subcategory         | NVARCHAR(50)  | Более подробная классификация продукта в рамках категории, например, по типу продукта.                                    |
| maintenance_required| NVARCHAR(50)  | Указывает, требуется ли техническое обслуживание изделия (например, "Да", "Нет").                                         |
| cost                | INT           | Себестоимость или базовая цена продукта, измеряемая в денежных единицах.                                                  |
| product_line        | NVARCHAR(50)  | Конкретная линейка продуктов или серия, к которой относится продукт (например, дорожный, горный).                         |
| start_date          | DATE          | Дата, когда продукт стал доступен для продажи или использования                                                           |

---

### 3. **gold.fact_sales**
- **Цель:** Хранит данные о транзакциях и продажах для аналитических целей.
- **Столбцы:**

| Имя столбца     | Тип данных     | Описание                                                                                     |
|-----------------|---------------|-----------------------------------------------------------------------------------------------|
| order_number    | NVARCHAR(50)  | Уникальный буквенно-цифровой идентификатор для каждого заказа на продажу (например, "SO54496")|
| product_key     | INT           | Суррогатный ключ, связывающий заказ с таблицей измерений продукта.                            |
| customer_key    | INT           | Суррогатный ключ, связывающий заказ с таблицей измерений клиента.                             |
| order_date      | DATE          | Дата, когда был размещен заказ.                                                               |
| shipping_date   | DATE          | Дата, когда заказ был отправлен клиенту.                                                      |
| due_date        | DATE          | Дата, когда должна была быть произведена оплата заказа.                                       |
| sales_amount    | INT           | Общая денежная стоимость продажи по данной позиции в целых валютных единицах (например, 25).  |
| quantity        | INT           | Количество единиц товара, заказанных для данной позиции (например, 1).                        |
| price           | INT           | Цена за единицу товара для данной позиции в целых валютных единицах (например, 25).           |
