# Data Warehouse
Этот проект, разработанный как портфолио-проект, демонстрирует комплексное решение для хранения данных


## Обзор проекта

Этот проект включает в себя:

1. **Архитектура данных**: Проектирование современного хранилища данных с использованием архитектуры Medallion **Бронзового**, **Серебряного** и **Золотого** уровней.
2. **ETL**: Извлечение, преобразование и загрузка данных из исходных систем в хранилище.
3. **Моделирование данных**: Разработка таблиц фактов и измерений, оптимизированных для аналитических запросов.

## Структура репозитория
```
traing_warehouse/
│
├── datasets/                           # Исходные наборы данных проекта
│                                       # Файлы с сырыми данными (CSV), используемые для загрузки
│                                       # в хранилище данных и запуска ETL-процессов
│
├── docs/                               # Документация проекта и архитектурные схемы
│                                       # Содержит визуальные и текстовые описания архитектуры
│                                       # хранилища данных, моделей данных и потоков данных
│
├── scripts/                            # SQL-скрипты для ETL и преобразования данных
│   │
│   ├── bronze/                         # Скрипты слоя Bronze
│   │                                   # Загрузка сырых данных во временные / staging-таблицы
│   │                                   # Минимальные преобразования, данные хранятся «как есть»
│   │
│   ├── silver/                         # Скрипты слоя Silver
│   │                                   # Очистка, нормализация и трансформация данных
│   │                                   # Подготовка данных для аналитического использования
│   │
│   └── gold/                           # Скрипты слоя Gold
│                                       # Формирование фактов и измерений
│                                       # Финальные аналитические модели данных (звёздная схема)
│
├── tests/                              # Скрипты тестирования и проверки качества данных
│                                       # Проверки целостности данных, дубликатов, NULL-значений
│                                       # и ссылочной целостности
│
├── README.md                           # Общее описание проекта и инструкции по использованию
│                                       # Цели проекта, архитектура и порядок запуска ETL-скриптов
│
├── LICENSE                             # Лицензия репозитория
│                                       # Определяет условия использования и распространения проекта


```
